How can one assess data visualization projects in an engaging manner?

This project was both an evaluation of Data Sketches, a data visualization project, and a website design challenge. It communicated ideas through a responsive grid system and simple graphical language.

Spring 2017. Data Visualization for Architecture, Urbanism and the Humanities. Taught by Juan Francisco Saldarriaga Chaux. In collaboration with Rachael Dottle, Eric Wong, Clara Dykstra and Andreea Seusan.
